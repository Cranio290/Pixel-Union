# Pixel Union:
A pixel game inspired by r/place

# Features:
- Full zoom
- Faction system
- Account system
- Chat
- Multiplayer
- Cooldown
- Canvas selection
- Moderation
- Leaderboard
- Pixel info

# Installation and info:
The game is free to use for everyone, you can check docs for help and information on the game and installation.
